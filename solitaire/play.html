About This Project

This is a faithful recreation of the classic Klondike Solitaire card game, built from the ground up with modern game development techniques. The project features a sophisticated AI player that demonstrates strategic play on the title screen, giving new players a glimpse of optimal strategies.

The game includes quality-of-life features like double-click quick moves to foundations, automatic card revealing, and an intelligent auto-complete system that takes over when victory is assured. The AI uses a priority-based decision tree to make human-like moves, avoiding pointless shuffling while maximizing the chance of winning.

All card movement logic, pile management, and game rules are implemented with clean, modular code. The collision detection system uses layered z-positioning and smart collider management to ensure cards respond correctly to clicks even when overlapping.
