About the Game

Crystalline Runner is a fast-paced neonwave endless runner inspired by lane-based classics like Subway Surfers. Dash through a shifting crystalline world, react at speed, and survive for as long as possible.

Controls
  • W / D – Switch lanes
  • Space – Jump

Technical Highlights

  • Built entirely in C# using Unity WebGL
  • Custom lane-based movement system
  • Procedural obstacle spawning
  • Optimized for browser performance
  • Stylized neonwave visual pipeline
